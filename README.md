## Instalação do Modelo

Primeiro faça download do repositório usando

```bash
  git clone
```
Após isso faça a instalação das bibliotecas PyTorch
```bash
  pip install torch
```
Após os procedimentos anteriores apenas digite
```bash
  python cliv.py
```
